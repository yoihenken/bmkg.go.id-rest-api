Documentation

BMKG API adalah Unofficial API dari web bmkg.go.id.
API Path : http://bmkg-scrap.herokuapp.com/api
Github : https://github.com/febryardiansyah/bmkg.go.id-rest-api

Request Description
/ Main Web Documentation
/api Not found route
/api/cuaca Weather in all Provinces of Indonesia
/api/cuaca/[province name] Weather in all cities with specific Provinces
/api/cuaca/bandara Weather in Indonesian Airport
/api/cuaca/dunia Weather in All Countries
/api/udara Air status in Indonesia
/api/bandara Weather in airplane indonesia
/api/gempa/terkini Latest earthquake in indonesia
/api/gempa/dirasakan Felt earthquake in indonesia
